# on repetition

***we call it new***  
***but it feels familiar***

Disney remakes the same stories. New actors. Cleaner visuals. Same structure.

On TikTok, one video works. Then everyone makes their version. Same audio. Same timing. Same reaction at the same second.

It is fun. It spreads fast. It feels like you are part of something.

But sometimes it feels like we are just repeating things with small edits.

The format already exists. The joke already landed once. We just place ourselves inside it.

It is easier that way.

Sometimes it feels like we are not creating as much as we are inserting ourselves into something already made.

Creating from nothing is slower. There is no template. No proof it will work.

Maybe we are slowly handing over the thinking.

Less building.  
More fitting in.

Less building.  
More rebuilding.
